# Free Canvas Logo

Free Canvas uses a combination mark built around an infinite workflow path inside a dark canvas tile.

## Files

- `free-canvas-logo.svg`: horizontal lockup for landing pages, headers, docs, and marketing.
- `free-canvas-mark.svg`: square mark for app chrome, avatars, favicons, and compact UI.
- `free-canvas-mark-mono.svg`: monochrome mark that inherits `currentColor`.
- `../favicon.svg`: optimized favicon copy of the mark.

## Design Rationale

The mark combines three product ideas:

- Infinite canvas: rounded dark tile and subtle grid.
- AI workflow: connected nodes and crossing paths.
- Creative generation: cyan, violet, and amber glow from the app color system.

## Color Tokens

- Canvas deep: `#050608`
- Surface: `#14171D`
- Border: `#2B313B`
- Cyan: `#28D7F5`
- Violet: `#9B7CFF`
- Amber: `#FFB454`
- Text: `#F5F7FA`
- Muted text: `#788493`

## Usage

- Keep clear space around the mark equal to at least 25% of its width.
- Use `free-canvas-mark.svg` at 24px and above.
- Use `free-canvas-logo.svg` at 140px width and above.
- Do not stretch, rotate, recolor the full-color logo, or place it on busy backgrounds.
