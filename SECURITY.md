# Security Policy

If you discover a security vulnerability, please do not open a public issue with exploit details.

Instead, contact the maintainer privately or use GitHub's private vulnerability reporting if it is enabled for this repository. Include:

- affected version/commit,
- reproduction steps,
- impact assessment,
- any suggested fix.

Before publishing this repository, rotate any secrets that may have existed in local files or git history (for example `.env*`, `.dev.vars`, `wrangler.toml`, API keys, OAuth client secrets, and Cloudflare resource credentials).
